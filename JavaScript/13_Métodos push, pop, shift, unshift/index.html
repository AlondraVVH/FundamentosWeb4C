Desafío de Arreglos en JavaScript

Ejercicio 1: Agregar elementos

Crea un arreglo vacío llamado compras y usa push() para agregar "pan", "leche" y "huevos".

Ejercicio 2: Quitar el último

Usa pop() para quitar el último producto del arreglo compras y muestra el valor eliminado.

Ejercicio 3: Agregar al inicio

Crea un arreglo colores = ["rojo", "azul"] y usa unshift() para agregar "amarillo" al inicio.

Ejercicio 4: Quitar el primero

Usa shift() para quitar el primer elemento del arreglo colores y muestra el valor eliminado.

Ejercicio 5: Combinación de métodos

Crea un arreglo numeros = [10, 20, 30], agrega 40 al final, quita el primero, agrega 5 al inicio y elimina el último.

Ejercicio 6: Agregar elementos en orden inverso

Crea un arreglo vacío orden y agrega los números 1, 2, 3 al inicio.

Ejercicio 7: Historial de mensajes

Crea un arreglo historial = [], agrega 3 mensajes con push(), elimina el último con pop() y muestra el mensaje eliminado.

Ejercicio 8: Cola de atención

Crea un arreglo cola = ["Cliente1", "Cliente2", "Cliente3"], elimina al primero con shift() y agrega "Cliente4" al final.

Ejercicio 9: Reordenar nombres

Crea un arreglo nombres = ["Pedro", "Juan", "Luis"], quita el último, agrega "Mateo" al inicio y "Ana" al final.

Ejercicio 10: Pilas y colas

Simula una pila y una cola usando push(), pop(), y shift().

Ejercicio 11: Completar la semana laboral

Arreglo inicial: ["lunes", "miércoles", "jueves"]. Agrega los días faltantes para completar la semana laboral.

Ejercicio 12: Registro de personas en una sala

Arreglo inicial: ["Carlos", "María", "Luis", "Ana", "Sofía"]. Modifícalo para obtener ["Pedro", "María", "Luis", "Camila"].

Ejercicio 13: Lista de tareas actualizada

Arreglo inicial: ["comer", "estudiar", "dormir", "jugar", "ver TV", "navegar"]. Modifícalo para obtener ["comer", "estudiar", "lavar la ropa", "organizar escritorio"].
